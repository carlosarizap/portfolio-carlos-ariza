## **Descripción General**

Desarrollo y optimización del sistema de cotización de seguros para **Sigue Seguro Corredores**, mejorando la experiencia del usuario y la eficiencia del proceso de cotización. El sistema permite comparar pólizas de múltiples aseguradoras en tiempo real.

## **Características Clave**

- **Interfaz de Usuario Optimizada**: Desarrollé un frontend moderno e intuitivo con **Next.js** y **React**, mejorando la experiencia de usuario y reduciendo el tiempo de carga.
- **Integración con Aseguradoras**: Implementé conexiones con **Mapfre, BCI y otras aseguradoras**, permitiendo la generación de cotizaciones en tiempo real.
- **Manejo de Datos**: Uso de **PostgreSQL** para almacenamiento estructurado y eficiente de información de cotizaciones y clientes.
- **Generación de Informes**: Creación de reportes personalizados para la toma de decisiones, con visualización de datos en tiempo real.

## **Tecnologías Utilizadas**

- **Next.js**: Para la optimización del frontend con SSR y CSR.
- **React**: Para la creación de componentes dinámicos y una mejor UX/UI.
- **PostgreSQL**: Para el manejo eficiente de datos.
- **Node.js**: Para el backend y la integración con APIs de aseguradoras.

## **Desafíos y Aprendizajes**

Uno de los principales desafíos fue la integración con múltiples aseguradoras, cada una con su propia API y estructura de datos. La solución fue desarrollar una **capa de abstracción** que permite manejar diferentes formatos de respuesta sin afectar la funcionalidad del sistema. También optimicé las consultas a la base de datos para mejorar la velocidad del sistema.

## **Resultados**

El sistema ha optimizado el proceso de cotización, reduciendo **un 40% el tiempo de respuesta** y mejorando la conversión de clientes. La integración con aseguradoras ha ampliado la oferta de pólizas disponibles, fortaleciendo la competitividad de **Sigue Seguro** en el mercado.
